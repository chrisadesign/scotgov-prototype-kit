The Scottish Government Prototype Kit

Warning
This is an unofficial, unsupported tool. It has not been created by an employee of any government department.

This kit lets you rapidly create HTML prototypes of Scottish Government services.

It’s a copy of the GOV.UK prototype kit with the Digital Scotland design system package installed and a layout setup.

About this page

We recommend you add links to your prototype to this page, rather than delete it. You'll find this page in the kit at '/app/views/index.html'.

You can change the service name by editing the file '/app/config.js'.
